Você pode usar qualquer versão do android studio que for acima da 1.5.
Se o seu computador tiver mais de 8GB de memoria RAM pode utilizar as versões atuais, se tiver menos memória é recomendado utilizar o android studio 3.5.

Android Studio 2025 https://developer.android.com/studio/install?hl=pt-br
SDK Sanbot http://blue.sanbotcloud.com:98/dev/docs/robot.html

As únicas mudanças serão na configuração da versão do gradle, plugin do gradle e appcompact.

Dentro do arquivo .zip do SDK terá o Sanbot_OpenSDK e navigationSDK.
Dentro destas pastas você encontrará arquivos .ARR, .jar e a documentação.

Sanbot_OpenSDK é para controlar o robô.
NavigationSDK é para executar rotas e mapas.
